Register a consent given to a specific person

Request Body schema: application/json
required
nin
required
string

Norwegian national identity number

scope_of_consent
required
string
Enum: "debt.unsecured.presentation" "debt.unsecured.processing"

The scope of consent (e.g. presentation))

consent_duration_days
required
integer <int32>

The consent duration (in days)

our_consent_id
required
string

The ID for the consent stored on your side. Must be unique

Debt Api

Get debt information with Access token granted in authorization code flow

When the access token is granted as a part of an OAuth2 Authorization code flow it contains information about the user and it is therefore enough to provide the access token to collect debt information

Authorizations:
None
header Parameters
Authorization
required
string
Example: Bearer osbj5aJNmLhzRH_IxdDdiKUycGFHncWxYVT9vsdWpy8.ViC0zK6sv83LOxmd2oIunMgRcSKgEe6O9IXymQ5kxww

Access token received from the auth server by using the authorization code flow

Responses

Preprod environment

https://api-preprod.norskgjeld.no/v1/debt

Production environment

https://api.norskgjeld.no/v1/debt
